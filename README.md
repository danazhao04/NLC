# New Long Cheng Restaurant Website

This is a simple static website for **New Long Cheng Chinese Restaurant**.

## Files

- `index.html`: page layout and business details
- `styles.css`: visual design and responsive behavior
- `script.js`: menu categories and item prices

## Update Phone, Address, or Hours

1. Open `index.html`.
2. Search for:
   - `(315) 410-2288` and `(315) 410-2228` for phone numbers
   - `1005 S Salina St, Syracuse, NY 13202` for address
   - `Monday - Thursday`, `Friday - Saturday`, `Sunday` for hours
3. Replace with your exact business details.

## Update Menu Items and Prices

1. Open `script.js`.
2. Edit the `menuData` array.
3. Each item uses this format:

```js
{ name: "General Tso's Chicken", price: "$14.95" }
```

4. Save and refresh the page.
